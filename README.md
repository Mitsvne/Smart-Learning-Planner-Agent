# Smart-Learning-Planner-Agent
一款具备多工具调用的 AI 学习规划 Agent。用户只需输入模糊的自然语言学习目标（如：3天入门Python），Agent 即可自主拆解任务、调度外部函数生成规划，并返回结构化的学习报告。
